# 🏠 House Price Prediction

A machine learning project that predicts house prices based on features like location, size, number of bedrooms, bathrooms, and property age using the King County House Sales dataset.

---

## 📂 Dataset
- **Source:** Kaggle — King County House Sales
- **Rows:** 21,613
- **Features:** 21
- **Target Column:** `price`

---

## 🔧 Steps Performed
1. **Data Cleaning** — Handled missing values, outliers, duplicates, and wrong data types
2. **Exploratory Data Analysis (EDA)** — Visualized price trends, correlations, and location-based patterns
3. **Feature Engineering** — Encoded categorical variables and created new features
4. **Model Training** — Trained multiple ML models and tuned hyperparameters
5. **Evaluation** — Compared models using MAE, RMSE, and R² score

---

## 🤖 Models & Results
| Model | R² Score |
|---|---|
| Linear Regression | 0.70 |
| Random Forest | 0.88 |
| XGBoost | 0.91 |
